
# Shipping Courier Website

Full-stack shipping courier service website with:
- Shipment tracking
- Admin dashboard
- Order management
- Customer contact form
- Responsive frontend
- MongoDB database
- JWT authentication

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Styling: CSS

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `.env` inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

## Features
- Live shipment tracking
- Shipment status updates
- Admin login
- Create shipment
- Delivery history
- Mobile responsive UI
